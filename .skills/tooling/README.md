# Tooling & Utilities Skills

Development tools, automation, dan utility skills.

## 📚 Available Skills

### Development & Planning
- **brainstorming**: Creative planning sebelum implementation
- **create-plan**: Plan & task breakdown
- **audit-website**: Website audits (SEO, performance, security)
- **understand-diff**: Git diff analysis
- **troubleshoot**: Debug chat agent behavior

### Project Management
- **project-setup-info-local**: Project scaffolding & setup
- **agent-customization**: Agent config & customization

### GitHub Integration
- **summarize-github-issue-pr-notification**: PR/Issue summary
- **suggest-fix-issue**: Issue fix suggestions
- **form-github-search-query**: GitHub search query builder
- **show-github-search-result**: Search results display
- **address-pr-comments**: PR review handling
- **create-pull-request**: PR creation
- **fix-customization-evaluation-diagnostics**: Customization fixes
