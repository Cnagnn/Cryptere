# Skills Quick Start

Panduan cepat menggunakan skills folder di Crypter project.

## 📍 Skills Folder Location

```
.skills/
```

Akses: `.skills/README.md`

## 🎯 5 Langkah Mulai

### 1. Explore Kategori
```
.skills/
├── laravel/          ← Backend
├── frontend/         ← UI/Components
├── testing/          ← QA
├── architecture/     ← Analysis
└── tooling/          ← Tools
```

### 2. Baca Overview
- [x] `.skills/README.md` - Full documentation
- [ ] `.skills/MANIFEST.md` - Lengkap listing
- [ ] `.skills/INDEX.md` - Quick reference

### 3. Pilih Skill yang Relevan
```bash
# Contoh: Buat authentication feature
Pilih:
  - laravel/fortify-development
  - laravel/wayfinder-development
  - frontend/inertia-react-development
```

### 4. Activate di Chat
```
@skill:laravel/fortify-development
@skill:frontend/inertia-react-development
@skill:testing/pest-testing
```

### 5. Follow Guidance
- Skill activate otomatis
- IDE provide hints
- Agent guide implementasi

## 💡 Pro Tips

1. **Activate selalu dari relevant category**
   ```
   ✅ @skill:laravel/fortify-development
   ❌ @skill:fortify-development  (tanpa category)
   ```

2. **Baca category README dulu**
   - `.skills/laravel/README.md`
   - `.skills/frontend/README.md`

3. **Use INDEX.md untuk quick reference**
   - `.skills/INDEX.md` - activation patterns
   - Combinations untuk common tasks

4. **Check MANIFEST.md untuk complete list**
   - Lengkap listing semua skills
   - Status dan metadata

5. **Combine related skills**
   - UI work: shadcn + tailwindcss
   - Backend: laravel-best-practices + wayfinder
   - Test: pest-testing + understand-diff
