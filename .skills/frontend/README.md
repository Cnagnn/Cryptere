# Frontend Skills

Client-side development skills untuk React + Inertia.js + Tailwind CSS.

## 📚 Available Skills

### 1. **inertia-react-development**
Mengembangkan aplikasi Inertia.js v3 dengan React.
- React pages, forms, navigation
- `<Link>`, `<Form>`, `useForm`, `useHttp` hooks
- Deferred props, prefetching, optimistic updates
- Instant visits, polling
- SSR configuration

**Trigger**: Inertia pages, React components, React forms, Inertia routing

### 2. **tailwindcss-development**
Styling dengan Tailwind CSS v4.
- Responsive grid layouts
- Flex/grid structures (dashboards, sidebars)
- UI components (cards, tables, navbars, forms)
- Dark mode variants
- Spacing & typography

**Trigger**: Semua styling work, layout building, responsive design

### 3. **shadcn**
Manajemen shadcn/ui components.
- Add components dengan shadcn CLI
- Components.json registry
- Component composition
- Preset management
- Styling & customization

**Trigger**: shadcn/ui components, components.json, `shadcn init`

## 🎨 Recommended Workflow

1. **Planning**: Brainstorm design
2. **Components**: `shadcn` → Select & add components
3. **Styling**: `tailwindcss-development` → Apply Tailwind
4. **Logic**: `inertia-react-development` → Wire up interactions

## 🔗 Tech Stack

- React v19
- Inertia.js v3
- Tailwind CSS v4
- shadcn/ui
- TypeScript

## 📝 Usage

```
@skill:frontend/inertia-react-development
@skill:frontend/tailwindcss-development

```

## ⚡ Quick Tips

- Always use `shadcn` components (user preference)
- Bootstrap shadcn first if not configured
- Use Tailwind for all styling
- Inertia for data binding & routing
